# CityWeather

Um bot de Discord criado utilizando OpenWeather API e Discord API.
Hosteado no Heroku

## Comandos
- !clima [city]
- !weather [city]

[city] deve ser substituido por uma cidade valida sem acentos

## Stack e Modulos
- NodeJS 
- OpenWeather API
- Discord API
- axios: 0.24.0
- discord.js: 13.3.1
- dotenv: 10.0.0
